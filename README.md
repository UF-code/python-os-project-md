# Project OS
Monitoring the System Resources and Their Usage

## Tech Stack
> Python, PyQT5, PyQtGraph, Psutil

### Project Setup & How To Compile n Run
```
pip install pyqt5
pip install psutil
pip install pyqtgraph
pip install hurry.filesize
python <filename>.py
```
